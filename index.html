About

I'm Reysan Madamba, a Computer Software Development student from NAIT, focused on building modern web applications and real-world systems.

I primarily work on full-stack web development, creating applications that are not only functional, but structured around real business logic and workflows. My experience includes building systems like substitution scheduling platforms, where handling rules, edge cases, and usability is just as important as writing code.

My tech stack includes C# .NET & Blazor, Java (IntelliJ, Maven, WildFly), HTML, CSS, JavaScript, and databases such as SQL, Firebase, and MongoDB. I also use tools like Python and LINQPad for scripting, testing, and data handling.

I enjoy working on complex problems and turning them into clean, usable solutions — building systems that are practical, scalable, and actually make people's work easier.
